# Specs (inner checkout)

When this repository is used as a **submodule** inside **Reflective Fold Obstruction**, the authoritative program EPICs live in the **outer** tree:

`../specs/IN-PROCESS/` (e.g. `SPEC_001_RFO` … `SPEC_004_RFO`).

Narrative design notes: `../specs/NOTES/PROJECT_VISION.md`, `../specs/NOTES/TWO_MODULE_MAP.md`.

If you cloned **only** `reflective-fold-obstruction-lean`, open or clone the outer workspace to work with those specs.
